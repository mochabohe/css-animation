# CSS Animation — 一站式 CSS 动画资源平台

## 项目背景

在日常项目开发中，动画环节长期存在两大痛点：

**一是 UI 与前端之间的"动画鸿沟"。** UI 设计师不清楚前端能实现哪些 CSS 动画效果，往往凭感觉提需求；前端开发者也难以准确还原设计师脑中的动效意图，双方只能通过文字描述或参考视频反复沟通，效率低、落地一致性差。缺少一个"所见即所得"的动画能力共识平台，是造成这一问题的根本原因。

**二是团队缺乏系统化的动画代码资产。** 每次遇到动画需求，开发者都需要从头搜索零散的代码片段，手动调试参数，完成后代码也难以沉淀复用，团队的动画经验始终停留在个人层面。

基于以上观察，发起 CSS Animation 项目——搭建一个可视化的 CSS 动画知识库，让 UI 设计师可以直接浏览效果、选择动画，前端开发者可以一键获取代码、调整参数、导出到目标框架，从根本上消除双方的信息不对称，同时为团队沉淀可复用的动画代码资产。

## 核心能力

### 1. 动画资源库 — UI 与前端的共同语言

平台收录了 **76 个精选 CSS 动画**，覆盖交互按钮、加载反馈、文字特效、背景氛围、运动效果、大数据可视化等 **8 大分类**。每个动画都有实时运行的可视化预览，UI 设计师可以直接看到效果并指定"就要这个"，前端开发者可以一键复制对应的完整代码——双方终于对着同一个画面沟通。

### 2. 代码实验室 — 在线调试、所见即所得

支持在线编辑 HTML 和 CSS 代码，修改即时生效；同时提供**参数化调节**——通过滑块调整动画时长，通过下拉框切换缓动函数。UI 和前端可以坐在一起，当场拖动参数确认效果，彻底告别"做出来和想的不一样"。

### 3. AI 智能功能矩阵 — 降低门槛、提升效率

集成了 **9 项 AI 能力**，是本项目的核心差异化亮点：

| 功能        | 说明                                                          |
| ----------- | ------------------------------------------------------------- |
| AI 智能搜索 | 输入自然语言描述（如"圆形旋转"），语义理解匹配相关动画        |
| AI 动画生成 | 在 AI 工坊中对话式描述需求，自动生成完整 CSS 动画代码         |
| AI 动画改造 | 对已有动画输入改造指令（如"把按钮改为红色"），AI 自动修改代码 |
| AI 动画解读 | 一键生成动画原理的中文详解，帮助开发者理解和学习              |
| 意图调参    | 点击"更快""更弹性""更柔和"等按钮，AI 自动推算最佳参数         |
| 框架导出    | 一键将动画转换为 React / Vue / Tailwind 代码格式              |
| 多轮对话    | AI 工坊支持上下文连续对话，逐步优化动画效果                   |

### 4. 主题与外观

支持 **11 种预设主题色 + 自定义颜色**，深色/浅色模式一键切换，方便在不同品牌调性下预览动画实际效果。
